Marker XML overview

As you're already aware, FusionMaps XT lets you create your custom defined markers on the map, using the Visual Map Builder. You can define as many markers for each map and then apply them.

For your ease, we've created marker definitions of important cities for most maps in the suite and provided them as XML/JSON to you. The XML/JSON files for each marker definition can either be accessed using the menu on the left, or they can be found in Download Package > MarkerData folder.

As you can see above, all the markers have been defined and applied in each XML.

If you wish to now show a marker in your map, you have two options:

  • If you never wish to use that particular marker, you can delete both the definition and application of the marker.
  • If you think that you might need it for some future map, just delete the application. The definition will continue to persist, but since you've removed the application, it won't show on the map.

For more information on how to use markers, customize them and connect them using connectors, please see the section How to use FusionCharts > Markers.
