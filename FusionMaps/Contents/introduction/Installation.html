Installing FusionMaps XT

Installation of FusionMaps XT merely involves copying and pasting the JavaScript and SWF files from the package into any of your folders.

Installing FusionMaps XT for your web application

Installation of FusionMaps XT for your web application is just a two-step affair:

  1. Create a folder named Maps in the root of your application (though, it's not mandatory to name the folder as Maps or create it under root folder. However, it organizes things a lot more, as all the pages within your website can now access the common set of maps).
  2. Copy all the JavaScript and SWF files from Download Package > Maps folder and paste it in this folder.

Installation is complete. There's no more step involved - you're now ready to use FusionMaps XT in your web application.
Installing FusionMaps XT on your local machine for general use

If you want to create maps on your local machine, follow the steps below:

  1. Create a folder named Maps inside your working folder (note that, it's not mandatory to name the folder as Maps or create it as a new folder. However, doing this way organizes things a lot more).
  2. Copy all the JavaScript and SWF files from Download Package > Maps folder and now paste it in this folder.
  3. To enable communication between Flash maps and JavaScript (like updating data, retrieving data, printing, exporting, event handling etc.) in your local machine, you need to setup Flash Player Global Security settings.

Note that setting up of Flash Player Global Security Settings is only required when you show the maps from your local file system. This is not required when the maps are displayed from a server (either localhost or remote).
Click here to learn more on how to configure Flash Player security settings »

You can manipulate the settings from the Global Security Settings panel.

If the version of your Flash Player is 10.3 or above you will need to right click on the map and select Global Settings... option from the right-click menu and do the following:

  1. Select Advanced tab
  2. Click Add to Trusted Zone button
  3. Add all the folder which contain the map SWF files to this zone

To get a detailed step-by-step tutorial on the above, please click here.

If the version of your Flash Player is lower than 10.3, it can be done in five simple steps:

  1. Go online to Adobe Flash Player Settings Manager and choose the Global Security Settings panel
  2. Select the "Always allow" radio button on the top to allow security exceptions. This control is marked as "1" in the image below
  3. From the " Edit Locations..." drop down box, select "Add location..." option to bring up a dialog box. The drop down box is marked as "2" in the image below
  4. Now select Browse for folder... and browse to the location where you are keeping the map SWF files
  5. After confirming your selection you should see the path of the exception added in area "3" of the image below


In case you wish to do this settings offline, you can ease this process by running a script file from the folder where you have kept the map SWF files. Go to How to setup Flash Player Global Security Settings? page.

That completes the installation for FusionMaps XT. Now, all you need to do is build map data and HTML page, which we'll see in the next section.
