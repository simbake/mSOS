Download Package Contents

FusionMaps XT Download Package contains the following. Please note that depending on the license that you've purchased, a few items might not be there.
FusionCharts JavaScript Class
 

FusionCharts JavaScript Class files are present in Download Package > Maps folder. It takes care of the rendering and management of maps. It helps render JavaScript maps. It also provides a large set of APIs to work with.

FusionCharts JavaScript Class can be categorized as follows:

  • Core FusionCharts JavaScript Class: FusionMaps XT pack contains 4 JavaScript files: FusionCharts.js, FusionCharts.HC.js, FusionCharts.HC.Maps.js and jquery.min.js. FusionCharts.js is the core FusionCharts JavaScript class that helps you embed maps in your Web pages in a user friendly way and offers functionality like updating data, retrieving data, supporting multiple data formats, event handling etc. Additionally, it also helps in rendering JavaScript maps (automatically using FusionCharts.HC.js, FusionCharts.HC.Maps.js and jquery.min.js and the relevant JavaScript map definition file.
  • Individual JavaScript map definition files: The definition of the JavaScript flavor of each map is stored in individual JavaScript file. All map definition JavaScript file names have "FusionCharts.HC. " as prefix, followed by the names of the maps. For example, the the JavaScript map definition for the World map is FusionCharts.HC.world.js.
  • FusionChartsExportComponent.js allows control over the client-side Export Component and provides interfaces to link the client-side Export Component to the maps present in the web page
  • FusionCharts.jqueryplugin.js provides easy jQuery syntaxes to render maps

To render JavaScript maps, it is absolutely necessary to keep all the map definition files and the files of FusionCharts JavaScript Class in the same folder as FusionCharts.js. All map definition JavaScript file names have "FusionCharts.HC. " as prefix, followed by the names of the maps. JavaScript maps are rendered with the help of FusionCharts.HC.js, FusionCharts.HC.Maps.js, jquery.min.js and individual JavaScript map definition files. For example, the the JavaScript map definition for the World map is FusionCharts.HC.world.js. However, you do not need to refer to all the JavaScript files in your code. FusionCharts.js automatically does it for you. All the JavaScript map definition files are present in Download Pack > Maps folder.
Map Files (Ready to use maps)
 

All the maps SWF and JavaScript files are present in Download Package > Maps Folder. Whenever you need to create maps for a new web application, just copy these files and paste it in your application.
Visual Map Builder
 

We've provided a Visual Map Builder that helps you easily configure your map data, markers, configuration etc. This is present in Download Package > Tools > GUI folder.
Map Entity List Generator
 

We've provided a small JavaScript application that helps you generate the entity list (Long name, short name and ID) for each map as a CSV file. This application is present in Download Package > SSGenerator folder. Just run Index.html.
FusionCharts Data Format Conversion Tool
 

This is a simple tool that lets you convert FusionMaps XT XML format to JSON and vice versa. This tool is present in the Tools > FCDataConverter folder.
Flash Player Security Setup
 

Script files present in the Download Package > Tools > FlashPlayerSecuritySetup folder to configure Flash Player Global Security settings - offline.
Server Side Export Handlers (Image/PDF Saving Scripts)
 

The scripts that allow you to save maps as PDF/images at server-side are present in Download Package > ExportHandlers folder. Please refer to Exporting as Image/PDF section in documentation on how to use them.

Note: Starting FusionMaps XT v3.3.0, JavaScript maps can be exported and saved to a server disk. To do so, you need to configure your own server. The sever should support PHP and JAVA (1.3 or above). Also, you need to place the FusionCharts PHP export handler and Apache Batik SVG Rasterizer library in your server.
Sample Code
 

All the code that we refer to in this documentation, is present in Download Package > Code folder. These are ready to use code samples, that you can copy-paste and run. Just make sure that you copy the Maps along with the respective example, else you will get an empty screen with a never ending progress bar.
Map Source Code
 

If you have the source code license of FusionMaps XT, you can find the source code at Download Package > SourceCode folder.
Map Gallery
 

We've built a map explorer to view outlines of each map in both JavaScript and Flash flavours. You can access the gallery from here or open index.html in a Web browser from the Download Package > Gallery folder.
Marker XML
 

We've built marker XML/JSON files to show important cities in most of the maps in suite. You can use the same marker definitions to show the cities on your maps. These XML/JSON files are present in Download Package > MarkerData folder.

You can view the city markers in individual maps using a Marker Viewer from Download Package > Tools > MarkerViewer > index.html.
