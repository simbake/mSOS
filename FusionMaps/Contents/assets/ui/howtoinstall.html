How to Install

Installing FusionMaps XT is a simple copy-n-paste procedure for web applications.

  1. Open Maps folder where all the maps (Flash SWF and JavaScript files) are located
  2. Copy the entire Maps folder to the root of your web application

That's it! Now you are only 15 minutes away from your wow moment.
